# Data

Place `instagram_reach.txt` in this directory before knitting `instagram_analysis.Rmd`.

The dataset is not tracked in this repository. To reproduce the analysis, obtain the raw file and save it as:

```
data/instagram_reach.txt
```

The file is a tab/space-delimited text file with a header row and 100 observations across 7 variables:
`S.No`, `USERNAME`, `Followers`, `Caption`, `Hashtags`, `Hours.since.posted`, `Likes`.
